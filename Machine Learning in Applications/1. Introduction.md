# A few definitions

## Artificial intelligence
It's the capability of the machines to display cognitive abilities typically associated to human minds (e.g., taking decisions, learning, problem solving, etc.).
Any AI agent needs to be able to:
- **Perceive** the environment (i.e. receive data from the external world).
- **Take actions** to achieve some goal.

## Machine learning
Subfield of artificial intelligence that gives computers the ability to learn how to solve a specific problem without being explicitly programmed for it.
**Learning** = deriving knowledge from experimental data.

## Supervised learning
Supervised learning is a type of ML techniques where the model has some knowledge before starting the training session, which is provided by humans. Usually this kind of knowledge consists in labels and annotations.
Generally speaking, supervised learning can fall into these two categories:
- **Classification**: every piece of data will fall into a specific category (defined by the labels).
- **Regression**: the model looks for the function that best fits all the data (look picture below).
>[!warning]
>The function found with regression is supposed only to approximate the real data fit, it should not match it exactly.
>If we match exactly the training data we will obtain an **overfitted** model, which will result in a disaster.

## Unsupervised learning
Deals with unlabeled data or data of unknown structure.
The algorithm explores the inherent structure of the data, without the guidance of a labelled training set or reward function.
Typical applications:
- **Clustering** (a.k.a. unsupervised classification).
- **Dimensionality reduction**.

### Clustering
The instances are partitioned into a number of classes (clusters) based on the:
- Maximization of similarity of instances of the same cluster.
- Minimization of similarity of instances of different clusters.
